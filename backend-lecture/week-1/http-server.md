# HTTP Server

## 프로젝트 생성

* Http Client 참조

## 소스코드

* 'https://github.com/MarkYoo23/Toy_JavaHttp/pull/1'

## 프로그램 구조

* Listen
  * 포트 번호 지정
  * 클라이언트의 접속 대기

* Request
  * 클라이언트의 요청을 받음

* Processing
  * 요청을 해석 및 처리

* Response
  * 요청에 대한 응답을 보냄

* Close
  * 클라이언트와의 연결을 종료

## 결론

* HTTP Server를 구현하여, Framework의 동작을 이해함
* HTTP 프로토콜이 처리되는 방법을 이해함
