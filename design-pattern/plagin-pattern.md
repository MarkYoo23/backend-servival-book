## 동기

* 컴파일이 아닌 구성 중(런타임)에 클래스를 연결 할 필요가 있음.
* 애플리케이션 코드를 여러 런타임 환경에서 특정 동작에 따른 구현을 적용해 실행해야 하는 경우가 있음.

## 특징

* 분리 인터페이스 정의
* 구현 클래스를 플러그인으로 제공
* 플러그인 팩터리 연결 명령을 지정할 장소 명시(파일, DB 등)
