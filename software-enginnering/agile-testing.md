# 애자일 테스트

## 동기

* 비즈니스 전문가(도메인)

## 특징

* 비즈니스 전문가(도메인)가 관심을 가질만한(또는 관심을 가져야 하는) 용어로 설명할 수 있는 테스트
* 애자일 특징상 빠르게 테스트를 작성하고 실행할 수 있어야 함

## 역사

* 브라이언 메릭 (Brian Marick)가 2003년에 발표

## 실제 사용 사례

* 다른 브라우저는 Javascript를 다르게 구현하므로 우리 제품이 가장 중요한 브라우저에서 작동하는지 테스트
