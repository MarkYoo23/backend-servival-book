## 동기

* 테스트 코드를 먼저 작성하면, 테스트 코드를 작성하기 쉬워진다.

## 특징

* TDD Cycle
  * Red : 실패하는 테스트를 작성한다.
  * Green : 테스트를 통과하는 코드를 작성한다.
  * Refactor : 중복을 제거하고 코드를 개선한다.
